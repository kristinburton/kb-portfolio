Marquis Envoy Project

A comprehensive redesign of the Marquis help center platform, focusing on user experience and accessibility.

Technologies

Project Overview

Marquis needed a modernized help center that would improve user navigation and reduce support tickets. The solution involved a complete redesign of their documentation platform.

Challenge

The existing help center was difficult to navigate and lacked proper organization, leading to increased support tickets and user frustration.

Solution

Implemented a new information architecture and modern design system that prioritized user needs and content discoverability.
